# Analyzing 2000 Students' Grades Dataset

This project focuses on analyzing a dataset containing the academic grades of 2000 students using Python.
The goal is to extract meaningful insights, perform statistical analysis, and visualize student performance
using popular Python data analysis libraries.

## Objectives
- Understand the distribution of students' grades
- Analyze performance trends
- Identify patterns and outliers in the dataset
- Visualize results for better interpretation

## Libraries Used
- NumPy
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Data cleaning and preprocessing
- Descriptive statistics
- Data visualization (histograms, boxplots, correlation heatmaps)
- Performance comparison and insights
